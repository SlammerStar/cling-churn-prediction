Architecture & Methodology

Cling is a full-stack ML application built for production scale.

  • Backend: Python Serverless Functions deployed on Vercel.
  • Machine Learning: Scikit-learn, XGBoost, Optuna. SMOTE for class imbalance, CalibratedClassifierCV for reliable probabilities.
  • Frontend: Vanilla HTML/JS with CSS Glassmorphism design. Chart.js for data visualization.
  • Database: Supabase (PostgreSQL) for prediction tracking.
